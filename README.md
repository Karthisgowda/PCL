# AI-Enabled Adaptive Solar Energy Management Presentation

This repository contains a 10-slide HTML presentation for the published IJSRED paper:

**AI-Enabled Adaptive Solar Energy Management System for Portable Charging Devices: A Proposed System Architecture**

Authors: Karthik S Gowda, Harsh Lakhotia, Hanoch Jiji, Ibrahim Sarfaraz Mukadam

Publication details:

- Journal: International Journal of Scientific Research and Engineering Development
- Issue: Volume 9, Issue 2, March-April 2026
- Paper ID: IJSRED-V9I2P454
- Pages: 2435-2443
- Published link: https://www.ijsred.com/volume9-issue2-part46.html
- Online edition: https://www.ijsred.com/mar-apr-2026.html

## View locally

Open `index.html` directly in a browser, or run:

```bash
npm start
```

Use the left/right arrow keys or on-screen buttons to navigate the deck.
